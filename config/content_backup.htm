更新cmsimde的方法

1.先確認好最新的版本可從已作好的同學或老師的倉儲取的

2.將目錄切換至cmsimde

 通常為

cd tmp

cd wcm2021

cd cmsimde

3.再來先把新的東西pull下來

git pull

4.再將版本checkout成最新的版本

git checkout 版本號

基本有出現就能完成更新cmsimde了
